# GlowUp Skincare — Shopify POC

A fully functional Shopify storefront built as a proof-of-concept portfolio piece.

## Timeline
- **Phase 1** (Week 1): Store Foundation — theme, Liquid, custom sections
- **Phase 2** (Week 2): API & Integrations — tracking, webhooks, APIs
- **Phase 3** (Week 3): A/B Testing — Intelligems setup and test campaigns
- **Phase 4** (Week 4): QA & Polish — testing, bug fixes, documentation

## Tech Stack
- Shopify Liquid + HTML/CSS/JavaScript/jQuery
- Shopify REST/GraphQL API
- Intelligems (A/B testing)
- Facebook Pixel + GA4
- Node.js (webhook handler)
